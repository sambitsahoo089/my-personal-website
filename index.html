Group Projects

Project: SPENDSENSE

SPENDSENSE is a glassmorphism-styled expense tracker built with HTML, CSS, & Javascript that runs entirely in the browser using localstorage for complete data privacy.

For students, it simplifies budgeting by tracking limited allowances against daily habits like food deliveries and travel, helping to prevent mid-month deficits.

For employees, it offers instant clarity on recurring bills, balances monthly salary baselines, and flags lifestyle creep using dynamic canvas donut charts.

By turning dry financial logs into interactive progress bars, SPENDSENSE helps both groups monitor their personal cash flow and maintain financial discipline effortlessly.
